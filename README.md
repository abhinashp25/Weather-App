# 🌌 Aakash Weather — Liquid Glass Experience

A premium, high-performance weather application built with **React** and **Three.js**, featuring a "Liquid Glass" design system and advanced weather intelligence.

![Weather App Preview](https://via.placeholder.com/1200x600?text=Aakash+Weather+Liquid+Glass+UI) *(Add your own screenshot here!)*

## ✨ Key Features

- **Liquid Glass UI:** A custom design system built with vanilla CSS, featuring advanced backdrop filters, glassmorphism, and responsive layouts.
- **WebGL 3D Weather Particles:** Real-time 3D rendering of rain, snow, and thunderstorm effects using **Three.js**.
- **Live Satellite Background:** Dynamic mapping using world-wide satellite imagery with weather-aware gradient overlays.
- **✨ AI Weather Analyst:** Hyper-personalized clothing and activity recommendations based on temperature, wind, and Air Quality (AQI).
- **⏳ Time Travel Weather:** Compare current conditions with simulated historical data from 10, 25, or 50 years ago to see climate trends.
- **🌐 Multilingual Support:** Fully translated into **English**, **हिन्दी (Hindi)**, and **ଓଡ଼ିଆ (Odia)**.
- **🛰️ Live Radar:** Integrated real-time precipitation radar maps.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [OpenWeatherMap API Key](https://openweathermap.org/api) (Free Tier)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/aakash-weather.git
   cd aakash-weather
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   - Copy `.env.example` to a new file named `.env`:
     ```bash
     cp .env.example .env
     ```
   - Open `.env` and paste your OpenWeatherMap API key:
     ```env
     VITE_OPENWEATHER_API_KEY=your_actual_key_here
     ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
